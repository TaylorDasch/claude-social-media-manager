# WEEKLY CONTENT BATCH SOP
## Standardized weekly rhythm for content production
## Claude references this when building daily plans

---

## MONDAY — Plan & Prep

### Morning Block (Lead Gen First — Always)
- Lead gen power block (calls, follow-ups, FUB tasks)
- This is always first, every day, non-negotiable

### Content Block
- Select this week's video topic from priority queue
- Choose script formula from VIDEO-SCRIPT-FORMULAS.md
- Pull data from TEMPLE-TX-DATA-VAULT.md
- Write or generate script (run /produce if full pipeline)
- Prep Tuesday shot list (which setups: desk, drive-through, walk-and-talk?)
- Pull Deal of the Week property + run numbers in spreadsheet

### AEO Block
- Work from AEO priority queue: page builds, transcript-to-blog conversions, schema deployment
- Check content freshness scanner output for stale pages

---

## TUESDAY — Film Day

### Morning
- Shortened lead gen block (30 min max)

### Production (Primary Block)
- FILM: Primary video (Deal of the Week or channel video)
- FILM: 2-3 TikTok/Shorts clips (vertical, face-to-camera)
- FILM: Any B-roll needed for upcoming pages
- If batch filming: change shirt/jacket between every 2 videos
- Upload primary video to YouTube before end of day

### Post-Upload
- Verify upload quality and thumbnail
- Set as Premiere if Deal of the Week (Tuesday morning release)

---

## WEDNESDAY — Edit, Deploy, Build

### Morning Block (Lead Gen First)
- Lead gen power block

### Post-Production
- Edit Tuesday footage if not done day-of
- Download YouTube auto-transcript (~30 min after upload)
- Clean transcript (fix proper nouns: BSW, Fort Hood, neighborhood names, numbers)
- Run /transcript-to-blog on cleaned transcript

### Deployment
- Deploy blog page to AgentFire with video embedded
- Add VideoObject + FAQPage + Article schema
- Clear AgentFire cache
- Request indexing in Google Search Console
- Post GMB update (30-sec clip or text + link to new page)

### AEO Block (Remaining Time)
- Continue priority queue from Monday

---

## THURSDAY — Engage & Source

### Morning Block (Lead Gen — Investor Focus)
- Lead gen power block (investor outreach, BP engagement)
- Vendor/relationship drops (lenders, inspectors, title)

### Content Engagement
- Post YouTube Community tab content (poll, data drop, or BTS photo)
- Reply to YouTube comments on this week's video
- BP forum engagement (if investor topic this week)
- Reddit engagement (if relevant threads found in Monday scanner)

### Next Week Prep
- Source property for next week's Deal of the Week
- Run numbers, screenshot MLS data
- Identify any trending topics from opportunity scanner

---

## FRIDAY — Review & Plan

### Morning Block (Lead Gen — Close Loops)
- Lead gen power block (close the week's open conversations)

### Pipeline Review
- Pipeline review in FUB
- Weekly attribution review (which content generated leads? See CONTENT-TO-LEAD-ATTRIBUTION.md)
- Check Otterly.AI for AEO citation changes

### Planning
- Plan next week's content calendar (run /content-calendar)
- Schedule any remaining social posts for the weekend
- Update content priority queue
- Set Monday's AEO block tasks

---

## SATURDAY — Technical & Optimization

- Website optimization: page audits, schema validation, AgentFire updates
- Run content freshness scanner
- Score 1-2 pages against AEO rubric (run /audit when built)
- Technical improvements (internal linking, broken link fixes)
- Upload cleaned transcripts to older videos (batch 3-5 per session)
- Add chapter markers to older videos (batch 3-5 per session)

---

## SUNDAY — Review & Reset

- TikTok performance review (what worked, what didn't)
- Review YouTube analytics for the week
- Finalize next week's content calendar
- Set Monday AEO priority queue
- Prep Tuesday filming shot list
- Mental reset — don't grind on Sunday

---

## RULES

1. **Lead gen is always the first block of the day** (Mon-Fri). Content does not replace prospecting.
2. **Tuesday is sacred film day.** Do not schedule appointments during the production block.
3. **Every video gets a page within 48 hours of upload.** Don't let transcripts sit.
4. **One video per week minimum, two ideal.** Quality over quantity for long-form.
5. **5 TikToks per week** during growth phase (batch from Tuesday filming).
6. **Community post between every video upload.** Keeps the channel alive in subscriber feeds.
7. **Friday review is mandatory.** If you skip it, you lose attribution visibility.
