## CTA PLACEMENT RULES

1. **YouTube Long-form:** Mention CTA verbally in last 30 seconds + pin matching comment + link in description (top 3 lines)
2. **YouTube Shorts:** On-screen text CTA in final 3 seconds + description link
3. **TikTok:** Verbal CTA in final 3 seconds + DM keyword in caption + comment reply trigger
4. **Blog Posts:** Inline CTA after the most valuable section (not just at the bottom) + sticky sidebar or floating bar
5. **Newsletter:** One primary CTA per section, matched to segment persona
6. **GMB Posts:** Soft CTA with website link to matching page
