# Thumbnail Brief Generator

## Trigger
"thumbnail brief", "thumbnail concept", "thumbnail for", "thumb idea"

## Required Inputs
- Video title (or 3 title options — brief will match the selected title)
- Video topic / content pillar
- Key emotion or reaction to convey

## Step-by-Step Execution

### Step 1 — Analyze the Title
- Identify the core hook word/phrase in the title
- The thumbnail text must NEVER repeat the title — it should complement it
- Determine the emotional angle: curiosity, shock, excitement, authority, warning

### Step 2 — Apply the 60-30-10 Color Rule
- **60% dominant:** Background/environment (usually property photo, drone shot, or solid color)
- **30% secondary:** Taylor's face/body or key visual element
- **10% accent:** Text overlay and graphic elements

Brand color application:
- Midnight #1e293b for text backgrounds/bars
- Emerald #059669 for accent elements, arrows, highlights
- Snow #f8fafc for text
- Red #ef4444 for warning/negative thumbnails
- Amber #d97706 for money/deal thumbnails

### Step 3 — Design the Composition

**Layout options (pick the best for this video):**

**A. Split Frame** — Taylor on left (40%), property/data on right (60%)
- Best for: neighborhood tours, property showcases
- Taylor expression: pointing at or looking toward the right side

**B. Taylor Foreground** — Large Taylor face/upper body, text overlay
- Best for: opinion videos, market updates, warnings
- Taylor expression: match the emotion (surprised, serious, excited)

**C. Before/After or Comparison** — Two images side by side with VS or arrow
- Best for: comparison videos (Temple vs Killeen, rent vs buy)
- Taylor small in corner or absent

**D. Data Overlay** — Property photo with large numbers/stats overlaid
- Best for: deal analysis, market data, investment breakdowns
- Text large enough to read on mobile (minimum 72pt equivalent)

### Step 4 — Write the Thumbnail Text
- Maximum 3-4 words (BOLD, readable at mobile thumbnail size)
- Must NOT repeat the video title
- Use contrast: light text on dark, or dark text on light with outline
- Examples:
  - Title: "5 Best Neighborhoods in Temple TX" → Thumb text: "HIDDEN GEMS" or "$180K STARTER"
  - Title: "Don't Buy in Temple Until You Watch This" → Thumb text: "HUGE MISTAKE" or "AVOID THIS"
  - Title: "Temple TX vs Killeen — Full Comparison" → Thumb text: "WINNER?" or "THE DATA"

### Step 5 — Specify Taylor's Expression & Gaze
- **Expression:** Surprised, pointing, serious, excited, confused, or nodding
- **Gaze direction:** Always toward the text or key visual element (leads viewer's eye)
- **Wardrobe note:** Dark polo or button-up preferred. No busy patterns.
- **Background:** If shooting thumbnail separately, use solid dark or emerald backdrop

### Step 6 — Output the Brief

```markdown
# Thumbnail Brief — [Video Title]

## Composition
Layout: [A/B/C/D]
Taylor position: [left/right/center/corner/absent]
Taylor expression: [specific expression]
Taylor gaze: [direction — toward text/property/camera]

## Text Overlay
Text: "[3-4 WORDS MAX]"
Font: Bold sans-serif (Outfit or Impact)
Color: [hex code]
Position: [top-right / bottom-left / center / etc.]
Effect: [outline / drop shadow / background bar]

## Background
Primary image: [drone shot / property front / solid color / map]
Treatment: [slight blur / darkened / split / gradient overlay]

## Color Palette
- Dominant (60%): [color + hex]
- Secondary (30%): [color + hex]
- Accent (10%): [color + hex]

## Additional Elements
- [ ] Arrow/pointer: [yes/no, direction]
- [ ] Price tag graphic: [yes/no, amount]
- [ ] Location pin: [yes/no]
- [ ] Emoji/icon: [yes/no, which]
- [ ] Border/frame: [yes/no, color]

## Mobile Check
At 168x94px (YouTube mobile thumbnail size):
- Is Taylor's face clearly visible? [yes/no]
- Is text readable? [yes/no]
- Does it stand out against gray YouTube background? [yes/no]

## A/B Notes
Variant A: [current brief]
Variant B suggestion: [alternative approach for split testing]
```

## Quality Checks
- [ ] Thumbnail text ≤ 4 words
- [ ] Text does NOT repeat the video title
- [ ] 60-30-10 color rule applied
- [ ] Mobile readable at thumbnail size
- [ ] Taylor's gaze leads toward key element
- [ ] No banned words or salesy language
- [ ] Brand colors used correctly
- [ ] Expression matches video tone

## Brand Rules
- Never use stock photos — always real property photos or Taylor
- No clip art, no cheesy graphics
- Dark premium feel — matches templetxhomes.net aesthetic
- If using property photo, must be actual property from the video
