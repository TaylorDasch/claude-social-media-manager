# TikTok Script

Generate a TikTok script with 3 hook options, value payload, Mac Moment, and DM keyword CTA.

## Usage
`/tiktok-script [topic]`

## Arguments
- `$1` — Topic, concept, or video idea (required)

## Execution
1. Read `skills/tiktok-script/SKILL.md` for full TikTok script workflow
2. Read `governance/QUALITY-GATES.md` for banned words and platform rules
3. Read `data/hook-bank.json` for existing hooks (avoid reuse)
4. Read `reference/LEAD-MAGNET-MATRIX.md` for DM keyword CTA matching
5. Determine persona — TikTok is BUYERS/RELOCATORS ONLY (never investor content)
6. Generate 3 hook options (each ≤ 3 seconds)
7. Write value payload with 7-second pattern interrupts
8. Include Mac Moment placeholder (5-8 sec physical comedy beat)
9. End with soft DM keyword CTA (never "link in bio" alone)
10. Add 3-tier hashtag set
11. Save to `output/YYYY-WXX/tiktok/[slug]-script.md`
12. Mark hook as "used" in `data/hook-bank.json` if pulled from bank
