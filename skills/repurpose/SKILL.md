# Skill: Content Repurposer

## Trigger
User says: "repurpose", "adapt this for", "cross-post", "turn this into"

## Instructions

**Important:** TikTok content is original property tours filmed on-site (see /tiktok-script skill). TikTok is NOT a repurpose target — never convert YouTube or blog content into TikTok. Instagram Reels can be repurposed from YouTube Shorts.

### Step 1: Identify Source Content
- What is the original piece? (YouTube video, blog post, deal analysis)
- What platform was it created for?
- What pillar/topic does it cover?

### Step 2: Apply Framing Effect
From config: Single data point → two distinct pieces

- **Relocator Frame**: Community vitality, job stability, family impact
- **Investor Frame**: Macro-economics, rental demand, appreciation catalysts

### Step 3: Generate All Platform Versions

#### From YouTube Long-form → 6 Platforms
1. **Blog Post Outline** (2,000+ words)
   - H1 as question
   - H2s as questions
   - All data from video as text
   - FAQ schema section (5+ Q&As)
   - Internal links

2. **YouTube Short Script**
   - Best hook + top 3 data points + verdict
   - DM keyword CTA

3. **BiggerPockets Forum Post**
   - Data-heavy summary
   - Specific numbers
   - NO video links (BP doesn't allow)
   - Link to blog post instead

4. **Newsletter Deep Dive Content**
   - Extract the analysis for The Deep Dive section
   - Reframe for email audience

5. **GBP Post** (reference `skills/gmb-post/SKILL.md` for 4-week rotation)
   - Match video topic to the appropriate post type (Market Update / Listing Spotlight / Neighborhood Guide / Expertise Tip)
   - Entity declaration + 2+ citable data points + specific page link
   - Under 300 words, AI citation optimized

6. **Social Posts** (via Gemini Social Repurposer format)
   - Relocator frame version
   - Investor frame version

#### From YouTube Short → Instagram Reel
1. **Instagram Reel**
   - Same video file, different caption
   - Write keyword-dense caption (micro-blog style, 150-300 words). Front-load searchable keywords naturally (e.g., "homes for sale in Temple TX", "moving to Temple Texas", neighborhood names, price points)
   - Reduce hashtags to 3-5 MAX: 1 broad (#realestate), 1 niche (#TexasRealEstate), 1-3 local (#TempleTX #CentralTexas #[neighborhood])
   - Note: Instagram 2026 algorithm weights caption text over hashtags for discovery. DM shares are the #1 ranking signal for Reels.
   - Note: swap trending audio for IG trending audio

### Step 4: Save Output
Save all versions to `output/YYYY-WXX/repurposed/[original-slug]/`

#### From YouTube Long-form → Additional Platforms (NEW)

7. **YouTube Community Post**
   Reference `skills/community-post/SKILL.md` for format.
   - 3 variations (Data Poll, Scars & All Post, Pop Quiz)
   - Schedule between this video and the next upload
   - Include templetxhomes.net link

8. **LinkedIn Carousel Outline**
   Reference `skills/linkedin-carousel/SKILL.md` for format.
   - Only generate for investor or BSW medical topics
   - 5-7 slide outline for PDF carousel
   - Analytical, authoritative tone
   - Include persona-matched CTA from reference/LEAD-MAGNET-MATRIX.md

9. **FUB Email Drip Segment**
   - Reframe the content as a 150-200 word email snippet
   - Match to the relevant FUB Action Plan (Investor/BSW/Military/Luxury)
   - Can be inserted into the appropriate automated sequence
   - Soft CTA: reply keyword or link to full blog post

## Rules
- Every repurposed piece must feel native to its platform — not a copy-paste
- Instagram needs longer captions, YouTube Shorts needs search-optimized titles
- BiggerPockets gets NO video links — blog links only
- Always apply pillar-appropriate hashtag combos from config
- Framing Effect: if the content can serve both audiences, write both frames
- LinkedIn carousels only for investor/BSW content (skip for general lifestyle)
- Community posts should be scheduled between video uploads (not same day)
- Reference reference/LEAD-MAGNET-MATRIX.md for all CTAs
- TikTok is NEVER a repurpose target — it's original property tour content only
- Instagram captions must be keyword-dense (150-300 words, micro-blog style). Limit to 3-5 hashtags. Caption text is the primary discovery mechanism in 2026.
- Use "Fort Hood" not "Fort Cavazos" (name reverted July 2025)
- Use "buy-and-hold investors" not "turnkey"
